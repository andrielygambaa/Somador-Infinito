# ➕ Somador Infinito

🚀 Um projeto web interativo desenvolvido com **HTML, CSS e JavaScript** que permite ao usuário somar números continuamente e visualizar o resultado atualizado em tempo real.

Este projeto foi criado com o objetivo de **praticar lógica de programação, manipulação do DOM e interatividade em páginas web**, utilizando apenas tecnologias fundamentais da web.

---

# 🌐 Demonstração do Projeto

O **Somador Infinito** permite que o usuário insira números quantas vezes quiser.
A cada novo valor inserido, o sistema adiciona esse número à soma total e atualiza o resultado automaticamente na tela.

💡 Isso cria uma experiência simples, rápida e interativa para entender como funciona a manipulação de dados no navegador.

---

# 🧠 Conceitos Praticados

Este projeto trabalha diversos conceitos importantes do desenvolvimento web:

✨ **Manipulação do DOM**
Atualização dinâmica de elementos da página usando JavaScript.

🔢 **Lógica de programação**
Controle da soma acumulada conforme novos números são adicionados.

🎨 **Estilização com CSS**
Interface simples, organizada e agradável.

💻 **HTML5**
Estrutura da página.

🎨 **CSS3**
Estilização e layout da interface.

⚡ **JavaScript**
Lógica da aplicação e manipulação da soma.

---

# 🎮 Como Usar

1️⃣ Digite um número no campo de entrada
2️⃣ Clique no botão **Somar**
3️⃣ O número será adicionado ao total
4️⃣ O resultado será atualizado automaticamente

Você pode repetir esse processo **infinitamente**, somando quantos números quiser.
